28. 实现 strStr()

  • August 15, 2020
  • Leetcode

实现 strStr() 函数。

给定一个 haystack 字符串和一个 needle 字符串，在 haystack 字符串中找出 needle 字符串出现的第一个位置 (从0开始)。如果不存在，则返回  -1。

示例 1：

输入：haystack = "hello", needle = "ll"

输出：2

说明:

当 needle 是空字符串时，我们应当返回什么值呢？这是一个在面试中很好的问题。 对于本题而言，当 needle 是空字符串时我们应当返回 0 。这与C语言的 strstr() 以及 Java的 indexOf() 定义相符。

 

这道题个人感觉又快又简单肯定是双指针，但是作为一个萌新接触算法不久，还是想练一练怎么写kmp算法。希望能在练习中锻炼自己！ 


    class Solution {
        public int strStr(String haystack, String needle) {
            if (needle == null || haystack == null) return 0;
            int[] next = nextArray(needle);
            int i = 0, j = 0;
            //开始匹配，如果相同就各加1，不相同直接根据模式串next数组调整；
            while (i < haystack.length() && j < needle.length()){
                if (j == -1 || haystack.charAt(i) == needle.charAt(j)){
                    i++;
                    j++;
                }
                else j = next[j];
            }
            //匹配结束，如果j等于needle长度说明匹配成功，i往回跳j的长度等于匹配开始位置；
            if (j == needle.length()) return i - j;
            else return -1;
        }
        // 构造next数组
        public int[] nextArray(String needle){
            if (needle.length() == 0) return new int[]{-1};
            int n = needle.length();
            int[] next = new int[n];
            int k = -1;
            int j = 0;
            next[0] = -1;
            while (j < n - 1){
                if (k == -1 || needle.charAt(k) == needle.charAt(j)){
                    k++;
                    j++;
                    next[j] = k;
                }
                else k = next[k];
            }
            return next;
        }
    }
