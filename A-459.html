459. 重复的子字符串

  • August 24, 2020
  • Leetcode

给定一个非空的字符串，判断它是否可以由它的一个子串重复多次构成。给定的字符串只含有小写英文字母，并且长度不超过10000。

示例 1：

输入: "abab"

输出: True

解释: 可由子字符串 "ab" 重复两次构成。

示例 2：

输入: "abcabcabcabc"

输出: True

解释: 可由子字符串 "abc" 重复四次构成。 (或者子字符串 "abcabc" 重复两次构成。)

 

比较傻瓜式的比较方法，因为一个字符串是由好多个子串重复得来的，那么我的子串末尾肯定是我字符串末尾那个字符。 然后我子串长度一定是会小于整个长度的一半的，不然不能成倍。

好了，我们就先重头开始寻找跟最后一个字符一样，而且我这段长度能被整段长度整除的字符子串。 找到这样的子串之后，我们是不是就能把整个字符串分割成m段了呢？

之后是不是比较一下这m段是不是都一样，就可以得出我整个字符串是不是重复的呢？ 附上代码和运行结果！


    class Solution {
        public boolean repeatedSubstringPattern(String s) {
            int n = s.length();
            char last = s.charAt(n-1);
            for (int i = 0; i < n / 2; i++){
                if (s.charAt(i) == last && n % (i + 1) == 0){
                    int m = n / (i + 1);
                    int j = 1;
                    while (j <= m - 1){
                        if (s.substring(0, (i + 1)).equals(s.substring(j * (i + 1), (j + 1) * (i + 1)))){
                            j++;
                        }else break;
                    }
                    if (j == m) return true;
                } else continue;
            }
            return false;
        }
    }
