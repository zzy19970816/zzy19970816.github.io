剑指 Offer 59 - I. 滑动窗口的最大值

  • August 25, 2020
  • Leetcode

给定一个数组 nums 和滑动窗口的大小 k，请找出所有滑动窗口里的最大值。

示例 1：


输入: nums = [1,3,-1,-3,5,3,6,7], 和 k = 3
输出: [3,3,5,5,6,7] 
解释: 

  滑动窗口的位置                最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7

提示： 你可以假设 k 总是有效的，在输入数组不为空的情况下，1 ≤ k ≤ 输入数组的大小。

 

我们用一个链表——双头都可以进出的队列来记录我们数组中元素的下标。

在每次窗口移动的时候，把不属于这次窗口的队头元素先去了，然后把新的元素加进队尾。

加的时候，遇到比自己小的先从队尾排出，然后把要加的元素加进来，这样就维护了队头元素一定是当前窗口中最大的。然后把处理后的队头元素加入答案。继续上代码。


    class Solution {
        public int[] maxSlidingWindow(int[] nums, int k) {
            if (nums.length == 0) return new int[0];
            // the temp will store the number's position
            LinkedList temp = new LinkedList();
            int index = 0;
            int[] res = new int[nums.length - k + 1];
            for (int i = 0; i < nums.length; i++){
                // if the first element of temp is out of window, just remove it;
                if (!temp.isEmpty() && temp.getFirst() <= i - k){
                    temp.removeFirst();
                }
                // if the last element is smaller than current one, remove them.
                while (!temp.isEmpty() && nums[temp.getLast()] < nums[i]){
                    temp.removeLast();
                }
                //add the current one into list;
                temp.addLast(i);
                if (i >= k - 1){
                    res[index++] = nums[temp.getFirst()];
                }
            }
            return res;
        }
    }
